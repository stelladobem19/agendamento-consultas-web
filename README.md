# Agendamento de Consultas

Sistema web desenvolvido para facilitar o agendamento e a organização de consultas em unidades de saúde.

---

## Tecnologias
- HTML
- CSS
- JavaScript

---

## Funcionalidades
- Visualização da agenda (dia, semana e mês)
- Cadastro de consultas
- Definição de horário de início e término
- Edição e exclusão de consultas
- Interface simples e intuitiva

---

## Objetivo
Criar uma solução simples para organizar atendimentos em postos de saúde, melhorando a experiência de pacientes e profissionais.

---

## Contexto
Projeto desenvolvido na disciplina de Seminários III, durante a graduação em Ciência da Computação.

---

## Status
Projeto acadêmico finalizado.
